# ShowPR (Beta) - GitHub PR Dashboard

**ShowPR** is a powerful GitHub Pull Request Dashboard that helps developers showcase and manage their contributions effectively. With ShowPR, you can view, filter, and share your pull request history through a beautiful and intuitive interface.  

> ⚠️ **Note:** This is the **Beta version** of ShowPR. An extended edition with additional features exists but is **closed-source**.

## Features

- **GitHub Authentication**: Secure login with GitHub OAuth
- **PR Dashboard**: View all your pull requests (open, closed, merged) in one place
- **Advanced Filtering**: Filter PRs by repository, status, and more
- **Customizable Profile**: Create a shareable profile link to showcase your contributions
- **Embed Widget**: Add a dynamic PR showcase to your personal website or GitHub README
- **Statistics & Analytics**: Track your PR activity with beautiful charts and metrics
- **Responsive Design**: Works seamlessly on all devices
- **Dark/Light Mode**: Choose your preferred theme

## Tech Stack

- Next.js 13+ (App Router)
- TypeScript
- Tailwind CSS
- Radix UI
- NextAuth.js
- Supabase
- Chart.js / Recharts
- Sentry
